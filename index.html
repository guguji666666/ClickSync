OVERRIDE
本工具仅支持桌面版 Chrome / Edge（Chromium 内核） 浏览器，否则 WebHID API 无法工作。
#01

首次连接必须手动授权

点击“连接”按钮后，请在浏览器弹出的设备选择窗口中手动选中您的鼠标并点击“连接”，网页才能获得通信权限。已授权的设备下次可自动连接。

#02

支持设备列表

目前支持的品牌型号包括： 雷蛇（ 毒蝰V3Pro、毒蝰V4Pro。请退出雷云驱动使用 ）、 罗技（ GPW3、GPW4朱雀、GPW4鹰眼、小狗、GPW5雪豹 ）、 雷柏（ 二代 54L15 的 3950 系列鼠标 ）、 ATK（ 54L15 的 3950 系列鼠标 ）、 Ninjutso（ Sora V3 ）。 以上品牌的型号均为测试可用型号，其他型号不做保证。

#03

使用注意

1 使用罗技鼠标时，本驱动会自动开启板载内存模式，使板载配置生效；开启后 GHUB 将无法接管设备。如需使用 GHUB，请先关闭板载内存模式。不受支持的罗技型号可能因缺少板载配置，出现按键不可用等异常，请谨慎尝试；如已出现问题，可关闭板载内存模式，或通过 GHUB 重新写入默认配置。 2 使用雷蛇鼠标时，请先完全退出雷云驱动软件，否则雷云可能覆盖网页驱动配置，导致设置无法生效。

使用方法
1.

系统/驱动设置：关闭win“增强指针精确度”；关闭驱动的移动同步 / 直线修正 / 纹波修正（Angle Snapping / Ripple / Motion Sync 等）。

2.

准备轨道：建议用胶带贴一条直线导轨并标出起点与终点。

3.

输入基准鼠标 DPI1：输入你需要匹配的鼠标的DPI

4.

开始测鼠标1：点击“开始测鼠标1（3次）”。每次按住左键不松开后开始移动，推到终点后松开左键结束；共做 3 次。尽量匀速直推，避免回拉。

5.

开始测鼠标2：同样做 3 次；右侧记录中若某次偏差较大，可点“重测”替换该次。

使用方法
1.

点击右侧测量区域开始，并锁定光标。

2.

自然握姿左右来回滑动不少于 10 次（尽量保持你认为的“水平”）。

右键暂停，在左侧“结果”查看推荐角度与建议角度；需要继续就点击右侧测量区域。

4.
